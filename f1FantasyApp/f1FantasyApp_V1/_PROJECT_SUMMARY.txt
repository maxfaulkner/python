KEY FEATURES IMPLEMENTED
================================================================================

✅ MARKET-BASED DYNAMIC PRICING
   - Performance delta calculation (beat/miss expectation)
   - Market pressure calculation (pick frequency)
   - Weekly price updates
   - Formula: newPrice = oldPrice × (1 + perfDelta×0.15 + marketPressure×0.08)

✅ AUTOMATED RACE IMPORT
   - Monday 9am auto-import from Ergast API
   - 5 retries with staggered timing (9, 20, 34, 51 min)
   - Admin notification if all retries fail
   - Fallback to manual entry

✅ WEEKLY TEAM SELECTION
   - 100M budget per week (fresh, no rollover)
   - Pick 5 drivers + 1 constructor
   - Constructor valued at ~2.5x average driver
   - Teams lock Friday (FP1), unlock Monday (results posted)

✅ POINTS SYSTEM
   - Standard F1 points (25 for 1st, 18 for 2nd, etc.)
   - Constructor points = sum of both drivers
   - Season-long cumulative scoring
   - Leaderboard ranked by points, tie-broken by wins

✅ EMAIL NOTIFICATIONS
   - Admin alerts when race import fails
   - Weekly "pick your team" reminders
   - Optional weekly recap emails

✅ COMPLETE REST API
   - Authentication (register, login)
   - Team management (CRUD)
   - Pricing queries
   - Leaderboard generation
   - Admin functions
